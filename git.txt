                                                        ---git---

git clone-->exact copy to all local devices from remote repo
git push-->local workspace to remote repo(all changes made)
git pull-->changes made to remote repo to local devices  

local workspace----(git add -A)--->staging area--(git commit -m "message")-->local repo--->remote repo

git init-->initialize git
ls -A--->list all files
git log-->shows all the commit
git checkout commit id-->back to prev commit
git checkout master-->current commit
git status--changes made

---steps---

git init
git status
git add -A
git commit -m "first commit"
git remote add origin(paste)
git push origin master
